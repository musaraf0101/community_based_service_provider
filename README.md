# Community Based Service Provider

A web application designed to connect community members with local service providers efficiently. This platform allows users to book various services such as plumbing, electrical work, and other skilled trades through a user-friendly interface.

## Features

-   User registration and authentication
-   Browse and search for service providers by profession
-   Booking system to request services from providers
-   Admin dashboard to manage service providers and user requests
-   Responsive design for accessibility on different devices

## Technologies Used

-   Laravel Framework (PHP)
-   Blade Templating Engine
-   MySQL Database
-   Bootstrap CSS Framework
-   JavaScript for interactive UI elements
