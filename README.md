# Community Based Service Provider

Community Based Service Provider is a web-based platform developed to bridge the gap between community members and local service providers. This application enables users to easily find, book, and communicate with skilled professionals such as plumbers, electricians, and other tradespeople within their locality. The system aims to streamline service delivery by providing a convenient and trustworthy digital marketplace that benefits both customers and service providers.

The platform features user registration and authentication, service provider profiles, a booking system, and an administrative dashboard to manage users and services. Built using the Laravel framework with a responsive design, this project emphasizes usability, efficiency, and community engagement.

By facilitating direct access to reliable service providers, this project addresses common challenges such as unavailability, delayed services, and lack of transparency, ultimately contributing to improved local service experiences.

## Features

-   User registration and authentication
-   Browse and search for service providers by profession
-   Booking system to request services from providers
-   Admin dashboard to manage service providers and user
-   Responsive design for accessibility on different devices

## Technologies Used

-   Laravel Framework (PHP)
-   Blade Templating Engine
-   MySQL Database
-   Bootstrap CSS Framework
-   JavaScript for interactive UI elements

## Installation

### Clone the repository:

git clone https://github.com/musaraf0101/community_based_service_provider.git

### Navigate to the project directory:

cd community_based_service_provider

### Install dependencies using Composer

composer install
npm install

### Configure your .env file with database credentials and other settings.

### Run migrations to create the database schema:

php artisan migrate

### Start the development server:

php artisan serve
